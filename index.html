Session 2.2

Classic 서버 이미지 생성

및 검증 실습

내 서버 이미지 생성 단계

Step 1

서버 선택 및 정지

Classic 콘솔의 Server 메뉴에서 마이그레이션할 서버를 선택하고, 데이터 정합성을 위해 서버를 정지합니다.

Step 2

이미지 생성 요청

[서버 관리] > [내 서버 이미지 생성] 메뉴를 클릭하고 이미지 이름과 설명을 입력하여 생성을 요청합니다.

Step 3

생성 완료 검증

Server Image 메뉴에서 방금 생성한 이미지가 '운영중' 상태로 표시되는지 확인합니다.

Session 2.3

이미지 VPC 복제

및 VPC 서버 생성

Classic에서 VPC로 이미지 복제 흐름

Classic

내 서버 이미지

VPC

복제된 서버 이미지

VPC

신규 서버 생성

VPC 서버 생성 단계

1

Server Image 메뉴에서 복제할 이미지를 선택하고, [VPC로 복제] 버튼을 클릭합니다.

2

복제가 완료되면 VPC 콘솔의 Server > Server Image 메뉴로 이동하여 복제된 이미지를 확인합니다.

3

복제된 이미지를 선택하고 [서버 생성]을 클릭하여 VPC 서버 생성 절차를 진행합니다.

4

VPC, Subnet, 인증키 등 필요한 정보를 입력하고 서버 생성을 완료합니다.
