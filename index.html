문제가 되기 전에 먼저 발견하고 대응하기

네이버클라우드 모니터링 도구

Cloud Insight

서버, DB 등 클라우드 자원의 성능 지표(CPU, Memory 등)를 수집, 조회하고 감시 규칙을 설정하는 서비스입니다.

Cloud Log Analytics

애플리케이션 및 시스템 로그를 실시간으로 수집, 분석, 시각화하여 서비스 상태를 심층적으로 파악하는 서비스입니다.

주요 서비스 핵심 모니터링 지표

Server

  • CPU Usage: > 80% (5min)
  • Memory Usage: > 90% (5min)
  • Network I/O: Spike or Drop

Database

  • Active Connections: > Max Conn * 80%
  • Slow Queries: Count > 10
  • Storage Usage: > 85%
